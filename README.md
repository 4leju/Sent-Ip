# Sent-Ip
* Que es SENT-IP?   
* Un simple progama/script que a través de una IP y un puerto abierto tendrás acceso a su dispositivo 

# INSTALACION TERMUX
* apt update && apt upgrade -y
* apt install git -y
* git clone https://github.com/4leju/Sent-Ip
* cd Sent-Ip
* bash install

# USO
* python sent-ip.py
